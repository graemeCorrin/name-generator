# Name Generator

A Web Application for generating names.

## Requirements
* Python 3.7
* PostgreSQL 9.6

## Setup

**Install required modules into your virtual environment**
```
pip install -r requirements.txt
```

**Create new database**
```
psql
create database name_generator_dev
```

**Setup database**
```
manage.py db upgrade
```

## Running the app server

Run
```
manage.py runserver
```
